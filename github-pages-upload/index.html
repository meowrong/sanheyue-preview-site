空間與設備

娛樂相聚

夜晚回到三合院，朋友圍在客廳唱歌、打麻將、看電影。KTV、電動麻將桌、投影設備與 Netflix 都已準備好，讓一群人的旅行不只停留在白天，也延續到每個熱鬧的夜晚。

海島起居

包棟旅行最舒服的地方，是可以像在家一樣自由安排每一餐。彡禾悅備有廚房、冰箱、飲水機與洗衣機，也提供戶外烤肉空間。白天出門探索小琉球，晚上回來一起下廚、烤肉、聊天，把旅行過成幾天自在的生活。

毛孩友善

彡禾悅也有一位可愛的小主人，牠叫可樂。我們知道毛孩也是家人，所以歡迎旅人帶著牠們一起來小琉球旅行。在庭院裡曬太陽、散步、陪家人朋友一起度過假期，讓毛孩也成為這趟相聚時光的一部分。
